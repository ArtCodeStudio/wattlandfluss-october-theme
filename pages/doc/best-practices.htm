Best Practices

{# Sizes #}

Text blocks

If the text content of the design you implements have breaks, you should not write this {{'
'|escape}} breaks hard in your code. Instead you should look how much width this text has and define this in your style. For this the container class can be extended, scss/_containers.scss: 

{{'.container {
    // containers with text should be smaller
    &.container-text {
        max-width: 620px;
    }
}'|escape }}
